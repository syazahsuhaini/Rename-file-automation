# Rename-file-automation
Auto rename file whenever system detect file is downloaded

# Current features
- auto move the intended files into a specialized folder
- rename the files using a template name and based on its order

# How to run it in terminal
1. Change the terminal directory to where the script originated
2. Run command python [script name]
3. For moving files into a folder, run: python script1.py
4. For renaming files, run: python script2.py {month} {year}
